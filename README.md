# 🏠 StayPoint

StayPoint is a full-stack property rental platform that allows users to explore, add, edit, and manage rental property listings with a clean and responsive interface.

---

## 🔗 Live Demo

🌍 **Live Website:** https://staypoint-1n4o.onrender.com/

> 💡 The application may take a few seconds to load if the Render server is inactive.

---

## 📖 Overview

Features include:

- 🔐 User Authentication
- 🏡 Property Listings
- ➕ Add, Edit & Delete Listings
- 🖼️ Image Uploads
- ⭐ Reviews & Ratings
- 📱 Responsive Design

---

**Backend**

**Frontend**
